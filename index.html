Quick install

Install QGIS on Windows in 4 steps

  • 1

    Download osgeo4w-setup.exe

    Click the download button above. This is the official OSGeo4W Network Installer from osgeo.org.

  • 2

    Run as Administrator

    Right-click the installer → Run as administrator. Choose Express Desktop Install.

  • 3

    Select QGIS package

    Choose qgis-ltr for the stable long-term release or qgis for the latest version. Accept all dependencies.

  • 4

    Launch QGIS

    Find QGIS Desktop in the Start menu. First launch takes 15–30 seconds while Python and GDAL initialise.

Quick answers

Common questions

LTR or Latest — which should I install?
For most users: LTR (Long Term Release). It is stable, well-tested and supported for at least one year. Choose Latest only if you specifically need a feature introduced in the last few months. See the full LTR vs Latest comparison.
What are the minimum system requirements for QGIS?
Minimum: 4 GB RAM, 2 GB disk space, Windows 10 64-bit. Recommended: 8+ GB RAM, SSD, dedicated GPU for large raster datasets. See full requirements guide.
Is QGIS free for commercial use?
Yes. QGIS is released under the GNU GPL licence, which permits free use for personal, academic and commercial purposes. There are no licensing fees or seat limits.
