Install LTR

Install QGIS LTR on Windows

  • 1

    Download and run osgeo4w-setup.exe

    Click the download button above. Right-click → Run as administrator. Choose Express Desktop Install.

  • 2

    Select qgis-ltr package

    In the package list, find and check qgis-ltr. This installs QGIS 3.40 LTR with all dependencies.

  • 3

    Complete install and launch

    Accept dependencies, choose a mirror, wait for download. Launch QGIS Desktop 3.40.5 from the Start menu.

FAQ

LTR download questions

What is the current QGIS LTR version in 2026?
QGIS 3.40.5 "Bratislava" is the current LTR as of mid-2026. It is the recommended version for most users, organisations and educational use.
How do I know when a new LTR is released?
QGIS releases a new LTR every October. Subscribe to the QGIS blog at blog.qgis.org for release announcements. In-app: Help → Check QGIS Version to see if an update is available.
Can I update LTR without losing my projects?
Yes. Updating from 3.40.3 to 3.40.5 (patch update) is safe — just re-run osgeo4w-setup.exe and it will update the packages. Your projects, data and settings are unaffected by patch updates.
