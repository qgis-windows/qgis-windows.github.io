FAQ

QGIS Windows FAQ — install, versions, errors & features 2026

Quick answers to the most common QGIS questions on Windows: download, LTR vs Latest, requirements, startup errors, Python issues and performance.

Download & install

Download & install

What is the latest QGIS version for Windows?
QGIS 3.42.1 (Latest) and QGIS 3.40.5 LTR (recommended stable). Download via OSGeo4W installer.
QGIS LTR or Latest — which should I install?
Most users should install LTR (3.40) for stability. Install Latest only if you need a specific new feature. See LTR vs Latest guide.
What are the minimum system requirements for QGIS?
Windows 10 64-bit, 4 GB RAM (8 GB recommended), 2 GB disk space. See full requirements guide.
How do I install QGIS without internet?
Download the standalone MSI installer (~1 GB) from qgis.org on a connected machine, transfer it, and install offline. See standalone installer guide.
Versions & updates

Versions & updates

What is QGIS LTR?
Long Term Release — one QGIS version per year designated for stability. Receives bug fixes for 12+ months, no new features. Currently QGIS 3.40 Bratislava.
Current QGIS LTR version in 2026?
QGIS 3.40.5 Bratislava is the current LTR as of mid-2026.
How do I update QGIS on Windows?
Re-run osgeo4w-setup.exe — it detects installed packages and offers updates. See update guide.
Can I have LTR and Latest installed at the same time?
Yes. In OSGeo4W setup, select both qgis-ltr and qgis. Both appear in the Start menu independently.
Errors & crashes

Errors & crashes

QGIS not opening on Windows
Hold Shift while launching to disable plugins. If it opens, a plugin is the cause. See not opening fix guide.
QGIS crashes on startup
Disable plugins (Shift on launch), or delete the QGIS profile at %APPDATA%\QGIS\QGIS3\profiles\default. See crash fix guide.
Python error: No module named qgis
Reinstall QGIS via OSGeo4W. Never run QGIS with the system Python. See Python error guide.
QGIS is very slow on Windows
Use SSD storage, build spatial indexes, use GeoPackage instead of Shapefile, and build raster overviews. See performance guide.
Data & features

Data & features

Layers not aligning on the map
Your layers have different CRS. Check each layer's CRS in Properties → Source. Set the project CRS to match. See CRS guide.
How to install QGIS plugins?
Plugins → Manage and Install Plugins → search → Install Plugin. No external download needed. See plugins guide.
Is QGIS free for commercial use?
Yes. QGIS is released under the GNU GPL licence — free for personal, academic and commercial use with no seat limits.
QGIS vs ArcGIS — which to choose?
QGIS is free and open source. ArcGIS Pro is a paid commercial product. For most GIS work, QGIS is fully capable. See full comparison.
